താരതമ്യ പഠനം

ആസൂത്രണകമ്മീഷൻ

അധ്യക്ഷൻ : ചെയർമാൻ (പ്രധാനമന്ത്രി)
ഭരണസമിതി : നാഷണൽ ഡെവലപ്മെന്റ് കൗൺസിൽ (NDC)പ്രധാനമന്ത്രി,കേന്ദ്രമന്ത്രിസഭ,മുഖ്യമന്ത്രിമാർ,സംസ്ഥാന ധനകാര്യവകുപ്പ് മന്ത്രിമാർ
ഉപാദ്ധ്യക്ഷൻ- : ഡെപ്യൂട്ടി ചെയർമാൻ
ഉപാദ്ധ്യക്ഷനെ നിയമിക്കുന്നത് : കാബിനറ്റ്
അനൗദ്യോഗിക അംഗങ്ങൾ : ആസൂത്രണ മന്ത്രി
ഓഫീസ് മാനേജ്മെന്റ് : ആസൂത്രണ സെക്രട്ടറി

നിതി ആയോഗ്

അധ്യക്ഷൻ : ചെയർമാൻ(പ്രധാനമന്ത്രി)
ഭരണസമിതി: ഗവേണിംഗ് കൗൺസിൽ(പ്രധാനമന്ത്രി,മുഖ്യമന്ത്രിമാർ,ലഫ് ഗവർണർമാർ,അംഗങ്ങൾ )-
ഉപാദ്ധ്യക്ഷൻ: വൈസ് ചെയർമാൻ
ഉപാദ്ധ്യക്ഷനെ നിയമിക്കുന്നത്: പ്രധാനമന്ത്രി
അനൗദ്യോഗിക അംഗങ്ങൾ : കേന്ദ്രമന്ത്രിസഭ
ഓഫീസ് മാനേജ്മെന്റ് : സി.ഇ.ഒ
