പത്തനംതിട്ട

കേരളത്തിലെ 13-ാമത്തെ ജില്ല?

ans : പത്തനംതിട്ട

പത്തനംതിട്ട ജില്ലയുടെ രൂപീകരണത്തിന് മുൻകൈയെടുത്ത വ്യക്തി ?

ans : കെ.കെ. നായർ

ഏറ്റവും കുറച്ച് റെയിൽപ്പാതയുള്ള ജില്ല?

ans : പത്തനംതിട്ട

പത്തനംതിട്ട ജില്ലയിലെ ഏക റെയിൽവേ സ്റ്റേഷൻ?

ans : തിരുവല്ല

പത്തനംതിട്ട ജില്ലയുമായി അതിർത്തി പങ്കിടുന്ന സംസ്ഥാനം?

ans : തമിഴ്നാട്

തെക്കൻ കേരളത്തിലെ ഏറ്റവും പ്രായം കുറഞ്ഞ ജി?

ans : പത്തനംതിട്ട

കടൽത്തീരമില്ലാത്ത ജില്ലകളിൽ തെക്കേയറ്റത്തുള്ള ജില്ല?

ans : പത്തനംതിട്ട

മൂഴിയാർ ഡാം സ്ഥിതിചെയ്യുന്ന ജില്ല?

ans : പത്തനംതിട്ട

കക്കാട് പദ്ധതി സ്ഥിതിചെയ്യുന്ന ജില്ല?

ans : പത്തനംതിട്ട

തീർത്ഥാടന ടൂറിസത്തിന്റെ ആസ്ഥാനം എന്നറിയപ്പെടുന്നത്?

ans : പത്തനംതിട്ട

പാർത്ഥസാരഥി ക്ഷേത്രം സ്ഥിതി ചെയ്യുന്നത്?

ans : ആറന്മുള (പത്തനംതിട്ട)

ജലത്തിലെ പൂരം എന്നറിയപ്പെടുന്നത്?

ans : ആറന്മുള വള്ളംകളി

ഉതൃട്ടാതി വള്ളംകളി (ആറന്മുള വള്ളംകളി) നടക്കുന്ന നദി?

ans : പമ്പ

നീളത്തിൽ മൂന്നാം സ്ഥാനമുള്ള നദി?

ans : പമ്പ (176 കീ.മി)

പത്തനംതിട്ട ജില്ലയിലെ ജലവൈദ്യുതപദ്ധതി?

ans : ശബരിഗിരി

ശബരിഗിരി പദ്ധതി സ്ഥിതി ചെയ്യുന്ന നദി?

ans : പമ്പ

മധ്യതിരുവിതാംകൂറിന്റെ ജീവനാഡി എന്നറിയപ്പെടുന്ന നദി?

ans : പമ്പ

എ.ഡി. 52-ൽ സെന്റ് തോമസിനാൽ നിർമ്മിക്കപ്പെട്ടു എന്ന് വിശ്വസിക്കുന്ന നിരണം പള്ളി സ്ഥിതിചെയ്യുന്നത്?

ans : പത്തനംതിട്ട

കേരളത്തിലെ താറാവു വളർത്തൽ കേന്ദ്രം?

ans : നിരണം

നിരണം കവികളുടെ ജന്മദേശം?

ans : പത്തനംതിട്ട

നിരണം കവികൾ?

ans : മാധവപണിക്കർ, ശങ്കരപണിക്കർ, രാമപണിക്കർ

ഇന്ത്യയിൽ സീറോ ജനസംഖ്യ വളർച്ചാനിരക്ക് കൈവരിച്ച ആദ്യ ജില്ല?

ans : പത്തനംതിട്ട

ജനസംഖ്യാവർദ്ധന നിരക്ക് ഏറ്റവും കുറഞ്ഞ ജില്ല?

ans : പത്തനംതിട്ട (-3.12%)

കേരളത്തിൽ ഏറ്റവും കൂടുതൽ റിസർവ് വനമുള്ള ജില്ല?

ans : പത്തനംതിട്ട

കേരളത്തിൽ സാക്ഷരത നിരക്ക് ഏറ്റവും കൂടുതലുള്ള ജില്ല?

ans : പത്തനംതിട്ട (96.93%)

ഇന്ത്യയിലെ ആദ്യ പോളിയോ വിമുക്ത ജില്ല?

ans : പത്തനംതിട്ട

പത്തനംതിട്ടയിലെ പ്രസിദ്ധമായ വെള്ളച്ചാട്ടം?

ans : പെരുന്തേനരുവി

മാരാമൺ കൺവെൻഷൻ നടക്കുന്ന ജില്ല?

ans : പത്തനംതിട്ട

ഏതു നദീതീരത്താണ് മാരാമൺ കൺവെൻഷൻ നടക്കുന്നത്?

ans : പമ്പ

ഏഷ്യയിലെ ഏറ്റവും വലിയ (ലോകത്തിലെ രണ്ടാമത്തെ) ക്രൈസ്തവ സമേളനം?

ans : മാരാമൺ കൺവെൻഷൻ

മാരാമൺ കൺവെൻഷൻ നടക്കുന്നത്?

ans : ഫെബ്രുവരിയിൽ

മാരാമൺ കൺവെൻഷൻ സംഘടിപ്പിക്കുന്നത്?

ans : മാർത്തോമാ ചർച്ച്

മാരാമൺ കൺവെൻഷൻ ആദ്യമായി നടന്ന വർഷം?

ans : 1895

കേരളത്തിലെ ഏറ്റവും വലിയ ഹിന്ദുമത സമ്മേളനം നടക്കുന്ന സ്ഥലം?

ans : ചെറുകോൽപ്പുഴ

ചെറുകോൽപ്പുഴ ഹിന്ദുമത സമ്മേളനം നടക്കുന്നത് ഏത് നദിയുടെ തീരത്താണ്?

ans : പമ്പ

ആറന്മുള കണ്ണാടി നിർമ്മിച്ചിരിക്കുന്നത്?

ans : ലോഹക്കൂട്ട് ഉപയോഗിച്ച്

വാസ്തുവിദ്യാ ഗുരുകുലം സ്ഥിതി ചെയ്യുന്നത്?

ans : ആറന്മുള

പടയണിക്ക് പ്രസിദ്ധമായ കടമ്മനിട്ട ദേവീക്ഷേത്രം സ്ഥിതി ചെയ്യുന്നത്?

ans : പത്തനംതിട്ട

ഓർമ്മപ്പെരുന്നാൾ നടക്കുന്ന പത്തനംതിട്ട ജില്ലയിലെ ക്രിസ്തീയ ദേവാലയം?

ans : മാർഗ്രിഗോറിയസ് ദേവാലയം

പത്തനംതിട്ട ജില്ലയുടെ ആസ്ഥാനം ഏതു തീരത്താണ്?

ans : അച്ചൻ കോവിലാർ

ദക്ഷിണ കുംഭമേള എന്നറിയപ്പെടുന്നത്?

ans : ശബരിമല മകരവിളക്ക്

ഇന്ത്യയിൽ സീസണിൽ വരുമാനം ഏറ്റവും കൂടുതലുള്ള ക്ഷേത്രം?

ans : ശബരിമല ക്ഷേത്രം

ശബരിമല സ്ഥിതി ചെയ്യുന്ന താലൂക്ക്?

ans : റാന്നി

കേരളത്തിലെ ഏറ്റവും വിസ്തൃതി കൂടിയ വനം ഡിവി ഷൻ?

ans : റാന്നി

വേലുത്തമ്പിദളവയുടെ അന്ത്യം കൊണ്ട് ചരിത്ര പ്രസിദ്ധമായ സ്ഥലം?

ans : മണ്ണടി (പത്തനംതിട്ട)

സരസകവി മുലൂർ പത്മനാഭപ്പണിക്കരുടെ സ്മാരകം സ്ഥിതി ചെയ്യുന്നത്?

ans : ഇലവും തിട്ട

ചെങ്ങറ ഭൂസമരം നടന്ന ജില്ല?

ans : പത്തനംതിട്ട

മണിയാർ ജലവൈദ്യുത പദ്ധതി സ്ഥിതിചെയ്യുന്നത്?

ans : പത്തനംതിട്ട

പത്തനംതിട്ട ജില്ലയിലെ ഏക ഹിൽസ്റ്റേഷൻ?

ans : ചരൽക്കുന്ന്

ദക്ഷിണ ഭഗീരഥി എന്നറിയപ്പെടുന്ന നദി ?

ans : പമ്പ

ബാരിസ് എന്നറിയപ്പെട്ടിരുന്ന നദി?

ans : പമ്പ

കേരളത്തിലെ ഏക പക്ഷിയോഗ നിർണ്ണയ ലാബ്?

ans : മഞ്ചാടി

മധ്യതിരുവിതാംകൂറിന്റെ ജീവരേഖ എന്നറിയപ്പെടുന്ന റോഡ്?

ans : T.K. റോഡ്(തിരുവല്ല കുമ്പഴ റോഡ്)

വർഷം മുഴുവൻ പൂത്തുനിൽക്കുന്ന കണിക്കൊന്നമരമുള്ള പത്തനംതിട്ടയിലെ പുരാതനക്ഷേത്രം?

ans : ആനിക്കാട്ടിലമ്മ ക്ഷേത്രം

ഗവി ഇക്കോ ടൂറിസം പദ്ധതി നടപ്പിലാക്കിയിരിക്കുന്ന ജില്ല?

ans : പത്തനംതിട്ട

ആനയുടെ മുഴുവൻ അസ്ഥിയും (288 എണ്ണം) പ്രദർശിപ്പിച്ചിരിക്കുന്ന കേരളത്തിലെ ഏക മ്യൂസിയം?

ans : ഗവി മ്യൂസിയം (കോന്നി)

ചിലന്തി ക്ഷേത്രം സ്ഥിതിചെയ്യുന്നത്?

ans : കൊടുമൺ

ആശ്ചര്യചൂഢാമണി രചിച്ച ശക്തിഭദ്രന്റെ ജന്മസ്ഥലം?

ans : കൊടുമൺ

ഗുരു നിത്യചൈതന്യയതിയുടെ ജന്മസ്ഥലം?

ans : വാകയാർ

മഹാത്മാ ഖാദി ആശ്രമം സ്ഥിതിചെയ്യുന്നത്?

ans : ഇലന്തൂർ (പത്തനംതിട്ട)

മഹാത്മാഗാന്ധിയുടെ നിർദ്ദേശമനുസരിച്ച്, മഹാത്മാ ഖാദി ആശ്രമം സ്ഥാപിച്ചത്?

ans : ടി.പി. ഗോപാലപിള്ള (1941)

ഖാദി തുണിത്തരങ്ങളുടെ വിപണനത്തിനായി ടി. പി. ഗോപാലപിള്ള രൂപീകരിച്ച പദ്ധതി?

ans : ഒരു പൈസാ നിക്ഷേപം
