കോന്നി ആനക്കൂട്

ആനക്കുട് സ്ഥിതി ചെയ്യുന്ന സ്ഥലം?

ans : കോന്നി

കോന്നി ആനക്കുട് സ്ഥാപിതമായത്?

ans : 1942

കോന്നി ആനക്കൂട് നിർമ്മിക്കുന്നതിനായി ഉപയോഗിച്ച മരം?

ans : കമ്പകം

കോന്നി ആനത്താവളത്തെക്കുറിച്ച് പ്രതിപാദിക്കുന്ന പുസ്തകം?

ans : ഐതിഹ്യമാല

പോർച്ചുഗലിന് ഭാരതത്തിന്റെ സമ്മാനമായി കോന്നി ആനക്കൂടിൽ നിന്നും നൽകിയ ആന?

ans : സംയുക്ത
