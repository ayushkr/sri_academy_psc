കൃതികൾ, കർത്താക്കൾ

അർത്ഥശാസ്ത്രം കൗടില്യൻ
അഷ്ടാധ്യായി : പാണിനി
ഇൻഡിക : മെഗസ്തനീസ്
മഹാഭാഷ്യം : പതഞ്ജലി
മുദ്രാരക്ഷസം : വിശാഖദത്തൻ
മൃച്ഛഘടികം : ശൂദ്രകൻ
ബുദ്ധചരിതം : അശ്വഘോഷൻ
പഞ്ചതന്ത്രം : വിഷ്ണുശർമ്മൻ
ബൃഹത്സംഹിത : വരാഹമിഹിൻ
സൂര്യസിദ്ധാന്തം : ആര്യഭടൻ
അമരകോശം : അമരസിംഹൻ
ദേവിചന്ദ്രഗുപ്ത വിശാഖദത്തൻ
സ്വപ്നവാസവദത്തം : ഭാസൻ
ഉത്തരരാമചരിതം : ഭവഭൂതി
കിരാതാർജ്ജുനീയം : ഭാരവി
ഋതുസംഹാരം : കാളിദാസൻ
ശിശുപാലവധം: മാഘ൯
വിക്രമാങ്കദേവചരിത ബിൽഹണൻ
രാജതരംഗിണി : കൽഹണൻ
പ്രിയദർശിക : ഹർഷവർധനൻ
രത്നാവലി : ഹർഷവർധനൻ
നാഗാനന്ദം : ഹർഷവർധനൻ
കാദംബരി : ബാണഭട്ടൻ
ഹർഷചരിതം : ബാണഭട്ടൻ
ഇന്ദ്രഭൂതി : ജ്ഞാനസിദ്ധി
രാവണവധം : -ഭട്ടി
ഫോക്കോക്കി : ഫാഹിയാൻ
സിയൂക്കി : ഹ്യൂയാൻസാങ്
മിതാക്ഷര : വിജ്ഞാനേശ്വര
ദശകുമാരചരിതം : ദണ്ഡി
മാലതിമാധവം : ഭവഭൂതി
മഹാവീരാഥരിത : ഭവഭൂതി
പൃഥ്വിരാജ്രാസോ : ചാന്ദ്ബർദായി
കവിരാജമാർഗം : അമോഘവർഷൻ
മിലിന്ദപൻഹ : നാഗസേനൻ
വാസവദത്ത : സുബന്ധു
നിഷാദചരിതം : ശ്രീഹർഷൻ
ഗീതഗോവിന്ദം : ജയദേവൻ
കഥാസരിത്സാഗരം : സോമദേവൻ
ബൃഹദ്കഥാമഞ്ജരി : ക്ഷേമേന്ദ്രൻ
സാഹിത്യരത്ന : സുർദാസ്
ബൃഹദ്കഥ : ഗുണാഡ്യ
സപ്തശോധക : ഹാലൻ
