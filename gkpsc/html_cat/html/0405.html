Bengal - Basic Info (West Bengal)

GKPSC EXAM HELPER.

Capital: Kolkata

Formation: 1956 November 1

Language: Bengali

Major Festival: Kali Pooja

Major Rivers:

  • Ganges
  • Damodar
  • Hoogly

State Animal: Fishing Cat

State Bird: White - breasted king fisher​

State Flower: Shephali

State Tree: Chatian

ബംഗാൾ (പശ്ചിമബംഗാൾ)

രൂപീകൃതമായത് -1956 നവംബർ 1

തലസ്ഥാനം : കൊൽക്കത്ത

ഹൈക്കോടതി : കൊൽക്കത്ത

ഔദ്യോഗികപക്ഷി ;- കിങ്ഫിഷർ

ഔദ്യോഗിക മൃഗം :- ഫിഷിങ് കൃാറ്റ്

ഔദ്യോഗിക പുഷ്ടം ;- പവിഴമല്ലി (ഷെഫാലി)

ഔദ്യോഗിക ഭാഷ :- ബംഗാളി, ഇംഗ്ലീഷ്

പ്രധാനഭാഷ - ബംഗാളി,ഇംഗ്ലീഷ്

ഔദ്യോഗിക നാമം :- ബംഗാൾ (ഇംഗ്ലീഷിൽ), ബംഗ്ല.(ബംഗാളിയിൽ) എന്ന് 2016 ആഗസ്തിൽ മാറ്റി.

പ്രധാന ആഘോഷം - കാളിപൂജ

പ്രധാന നൃത്തരൂപങ്ങൾ - ജാത്ര,കാഥി

പ്രധാന നദികൾ -

  1. ഗംഗ
  2. ദാമോദർ
  3. ഭഗീരഥി
  4. ഹുഗ്ലി

  2. സിംഗൂർ,
  3. ബാറ്റനഗർ,

  5. ഡാർജിലിങ് സുഖവാസ കേന്ദ്രം
