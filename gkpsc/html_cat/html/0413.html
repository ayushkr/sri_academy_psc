Rajasthan - Basic Info

Capital: Jaipur

Formation: 1956 November 1

Language: Rajasthani

Major Festivals: Gangaur, Marwar, Teej

Major Rivers:

  • Luni
  • Chambal
  • Banas

State Animal: Camel Chinkara

State Bird: Great Indian Bustard

State Flower: Rohira

State Tree: Khejri
