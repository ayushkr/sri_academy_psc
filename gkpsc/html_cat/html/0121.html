1. ‘എനിക്ക് രക്തം തരൂഞാൻ നിങ്ങൾക്കു സ്വാതന്ത്ര്യം നേടിത്തരാം’’ എന്നു പറഞ്ഞ നേതാവ്?

Ans : സുഭാഷ് ചന്ദ്രബോസ്

2. ഭാരത്മാതാ എന്ന പേരിൽ നാടകമെഴുതിയ ദേശാഭിമാനി?

Ans : കിരൺചന്ദ്രബാനർജി.

3. കാര്‍ ബാറ്ററിയില്‍ അടങ്ങിയിരിക്കുന്ന ആസിഡന്‍റെ പേര് എന്താണ്?

Ans : സള്‍ഫ്യൂറിക്കാസിഡ്

4. സാഹ ഇൻസ്റ്റിറ്റ്യൂട്ട് ഓഫ് ന്യൂക്ലിയർ ഫിസിക്സ്?

Ans : കൊൽക്കത്ത

5. പഞ്ചാബിലെ നൗജവാൻ ഭാരതസഭ എന്ന സംഘടനയ്ക്ക് രൂപം നൽകിയതാര് ?

Ans : ഭഗത് സിങ്

6. ദേശീയ ഉൾനാടൻ ജലഗതാഗതത്തിന്‍റെ (National Inland Navigation Institute - ( NINI) ആസ്ഥാനം?

Ans : പാറ്റ്ന

7. കണ്ണിനകത്ത് അസാമാന്യ മർദ്ദം ഉളവാക്കുന്ന വൈകല്യം?

Ans : ഗ്ലോക്കോമാ

8. സിന്ധു നിവാസികളുടെ പ്രധാന ആഹാരം?

Ans : ഗോതമ്പ്

9. ഏത് ബാങ്കിന്‍റെ മുദ്രാവാക്യമാണ് " ദ നേഷൻ ബാങ്ക്സ് ഓൺ അസ്"?

Ans : എസ്.ബി.ഐ

10. രക്തകുഴലുകളിൽ നിന്നും ഹൃദയത്തിലേയ്ക്ക് തിരികെ രക്തം ഒഴുകാതെ സഹായിക്കുന്ന വാൽവ്?

Ans : അർധചന്ദ്രാകാര വാൽവ് (Semilunar Valve )

11. ഇന്ത്യയിലെ സംസ്കൃത ഗ്രാമം എന്നറിയപ്പെടുന്നത്?

Ans : മാട്ടൂർ (കർണാടക)

12. ശ്രീഭട്ടാരകൻ എന്നറിയപ്പെട്ടത്?

Ans : ചട്ടമ്പി സ്വാമികൾ

13. അറബിക്കടലിന്‍റെ റാണി എന്നറിയപ്പെടുന്നത്?

Ans : കൊച്ചി

14. ബംഗാൾ കടുവ എന്നറിയപ്പെടുന്ന ക്രിക്കറ്റ് താരം?

Ans : സൗരവ് ഗാംഗുലി

15. ‘ചരകസംഹിത’ എന്ന കൃതി രചിച്ചത്?

16. സത്യത്തിന്‍റെ തുറമുഖം എന്നറിയപ്പെട്ടിരുന്നത്?

17. ശിവജിയുമായി നിരന്തരം യുദ്ധത്തിലേർപ്പെട്ട മുഗൾ ചക്രവർത്തി?

18. സംസ്ഥാന വിവരാവകാശ കമ്മിഷന്‍റെ ആസ്ഥാനം?
