Assam - Basic Info

Capital: Dispur

Formation: 1956 November 1

Language: Assamese

Major Festival: Bihu

Major River: Brahmaputra

State Animal: One horned Rhinoceros

State Bird: White winged wood duck

State Flower: Foxtail orchid

State Tree: Hollong
