Punjab - Basic Info

Capital: Chandigarh

Formation: 1956 November 1

Language: Punjabi

Major Festival: Lohri

Major Rivers:

  • Beas
  • Sutlaj
  • Ravi

State Animal: Blackbuck

State Bird: Northern Goshawk

State Tree: Sheesham

പഞ്ചാബ്

തലസ്ഥാനം -ചണ്ഡീഗഢ്

രൂപീകൃതമായത് - 1956 നവംബർ 1

പ്രധാന ആഘോഷം-ലോഹ്റി

പ്രധാന നൃത്തരൂപങ്ങൾ-ഭാംഗ,ഗിഡ

പ്രധാന നദികൾ -

  1. ബിയാസ്,
  2. സത്ലജ്,
  3. രവി,
  4. ചിനാബ്,
  5. ഝലം

ഹൈക്കോടതി :- ചണ്ഡിഗഢ്

ഔദ്യോഗിക പക്ഷി :- നോർത്തേൺ ഗോഷാവ്ക്

ഔദ്യോഗിക മൃഗം :- കൃഷ്ണമൃഗം (കരിമാൻ).

ഔദ്യോഗിക വൃക്ഷം :- ശിം ശപ(indian rosewood)

ഔദ്യോഗിക ഭാഷ :- പഞ്ചാബി

പഞ്ചാബിന്റെ സ്ഥാപകൻ :- ബന്ദാസിങ്ബഹദൂർ

പഞ്ചാബ്എന്ന പദത്തിന്റെ അർഥം :- അഞ്ചുനദികളുടെ നാട്
