ഇന്ത്യൻ സിനിമ

ലോകത്തിലെ ഏറ്റവും വലിയ സിനിമാ വ്യവസായം?
Ans : ഇന്ത്യൻ സിനിമ
ഇന്ത്യൻ സിനിമയുടെ പിതാവ്?
Ans : ദാദാ സാഹിബ് ഫാൽക്കെ
ഇന്ത്യയിൽ ആദ്യമായി സിനിമ പ്രദർശനം നടന്നത്?
Ans : 1896 ജൂലൈ 7 (മുംബൈയിലെ വാട്സൺ ഹോട്ടലിൽ വച്ച് )
വാട്സൺ ഹോട്ടലിന്റെ ഇപ്പോഴത്തെ പേര്?
Ans : Esplanade Mansion
ഇന്ത്യയിൽ ആദ്യമായി സിനിമ പ്രദർശനം നടത്തിയത്?
Ans : ലൂമിയർ സഹോദരന്മാർ (ദി സീ ബാത്ത്,എ ഡെമോളിഷൻ,ലിവിങ് ദി ഫാക്ടറി,അറൈവൽ ഓഫ് എ ട്രെയിൻ,ലേഡീസ് ആന്റ് സോൾജിയേഴ്സ്,എൻട്രി ഓഫ് സിനിമാറ്റോഗ്രഫി എന്നീ ആറ് നിശബ്ദ ചിത്രങ്ങൾ പ്രദർശിപ്പിച്ചു)
ഇന്ത്യൻ സിനിമാ മേഖലയിൽ നൽകപ്പെടുന്ന പരമോന്നത പുരസ്കാരം?
Ans : ദാദാ സാഹിബ് ഫാൽക്കെ പുരസ്കാരം
ദാദാസാഹിബ് ഫാൽക്കെ പുരസ്കാരം ഏർപ്പെടുത്തിയ വർഷം?
Ans : 1969
ദാദാസാഹിബ് ഫാൽക്കെ പുരസ്കാര തുക?
Ans : പത്ത് ലക്ഷം രൂപ
പുണ്ഡാലിക് സിനിമയുടെ നിർമ്മാതാവ്?
Ans : ദാദാ സാഹിബ് തോൺ (സംവിധാനം -എൻ.ജി.ചിത്രേ )
ഫാൽക്കെ പുരസ്കാരവും ഭാരതരത്നവും ലഭിച്ച വ്യക്തികൾ?
Ans : ലതാ മങ്കേഷ്കർ, സത്യജിത് റേ
പൂർണ്ണമായും ഇന്ത്യയിൽ നിർമ്മിച്ച ആദ്യത്തെ സിനിമ?
Ans : രാജാ ഹരിശ്ചന്ദ്ര
രാജാ ഹരിശ്ചന്ദ്രയുടെ നിർമ്മാതാവ്?
Ans : ദാദാ സാഹിബ് ഫാൽക്കേ
ആദ്യ ഇന്ത്യൻ ശബ്ദ ചിത്രം?
Ans : ആലം ആര (1931)
'ആലം ആര’ എന്ന ചിത്രം സംവിധാനം ചെയ്തത്?
Ans : അർദേശിർ ഇറാനി
ഇന്ത്യയിലെ ആദ്യ 70 എം.എം. ചിത്രം?
Ans : എറൗണ്ട് ദ് വേൾഡ് (1967)
പൂർണ്ണമായും ഇന്ത്യയിൽ നിർമ്മിച്ച ആദ്യ 70 എം. എം. ചിത്രം?
Ans : ഷോലെ
ഷോലെ സിനിമ സംവിധാനം ചെയ്തത്?
Ans : രമേഷ് സിപ്പി
ഷോലെ സിനിമ തുടർച്ചയായി അഞ്ചു വർഷം പ്രദർശിപ്പിച്ച മുംബൈയിലെ തിയേറ്റർ?
Ans : മിനർവ
ഇന്ത്യയിലെ ആദ്യ കളർ ചിത്രം?
Ans : ബിൽവാ മംഗൾ (1932) (ബ്രിട്ടണിൽ വച്ചാണ് പ്രോസസിംഗ് നടന്നത്)
പൂർണ്ണമായും ഇന്ത്യയിൽ നിർമ്മിച്ച ആദ്യ കളർ ചിത്രം?
Ans : കിസാൻ കന്യ (1937)
ഇന്ത്യയിലെ ആദ്യ ത്രീഡി ചിത്രമായ മൈഡിയർ കുട്ടിച്ചാത്തൻ സംവിധാനം ചെയ്തത്?
Ans : ജിജോ
ഏറ്റവും കൂടുതൽ കാലം പ്രദർശിപ്പിച്ച ഇന്ത്യൻ സിനിമ?
Ans : ദിൽവാലെ ദുൽഹനിയാ ലേ ജായേംഗേ (20 വർഷം തുടർച്ചയായി മുംബൈ, മറാത്ത മന്ദിർ തീയേറ്ററിൽ പ്രദർശിപ്പിച്ചു. 2015 ഫെബ്രുവരിയിൽ പ്രദർശനം അവസാനിപ്പിച്ചു) (സംവിധാനം - ആദിത്യ ചോപ്ര)
ആദ്യമായി പശ്ചാത്തലസംഗീതം ഉൾപ്പെടുത്തിയ ഇന്ത്യൻ സിനിമ?
Ans : ചാണ്ഡിദാസ് (സംവിധാനം - ദെബാക്കി ബോസ്)
‘ചാണ്ഡിദാസ്’ എന്ന സിനിമയുടെ സംഗീത സംവിധാനം നിർവഹിച്ചത്?
Ans : ആർ.സി.ബോറൻ

Ans : മുംബൈ

ഇന്ത്യയിൽ അന്താരാഷ്ട്ര ചലച്ചിത്രോത്സവം ആരംഭിച്ച വർഷം?
Ans : 1952
അന്താരാഷ്ട്ര ചലച്ചിത്രോത്സവത്തിൽ ഏറ്റവും മികച്ച ചിത്രത്തിന് നൽകുന്ന അവാർഡ്?
Ans : സുവർണമയൂരം

Ans : മുംബൈ

ഫിലിം ആൻഡ് ടെലിവിഷൻ ഇൻസ്റ്റിറ്റ്യൂട്ട് ഓഫ് ഇന്ത്യ പുണെയിൽ പ്രവർത്തിക്കുന്നു.
ഇന്ത്യയിൽ സിനിമകൾക്ക് പൊതുപ്രദർശനത്തിന് അനുമതി നൽകുന്നത് സെൻട്രൽ ബോർഡ് ഓഫ് ഫിലിം സർട്ടിഫിക്കേഷനാണ് (CBFC).

1896 ജൂലായ് 7-നു നടന്ന ഇന്ത്യയിലെ ആദ്യ സിനിമ പ്രദർശനത്തിൽ ആറ് നിശ്ശബ്ദ ഹ്രസ്വ ചിത്രങ്ങളാണ് ലൂമിയർ സഹോദരന്മാർ പ്രദർശിപ്പിച്ചത്. 'ദി സീബാത്ത്, അറൈവൽ ഓഫ് എ ട്രെയിൻ, എഡെമോളിഷൻ, ലേഡീസ് ആൻഡ് സോൾജിയേഴ്സ് ഓൺ വീൽസ്, എൻട്രി ഓഫ് സിനിമാട്ടോഗ്രാഫി, ലീവിങ് ദി ഫാക്ടറി എന്നിവ.

1870 ഏപ്രിൽ 30ന് മഹാരാഷ്ട്രയിലെ നാസിക്കിനടുത്ത് ത്രെെയംബകേശ്വറിൽ ജനിച്ചു.
ഒട്ടേറെ പ്രതിസന്ധികളെ തരണം ചെയ്ത് 1913 ൽ ആദ്യചിത്രം പൂർത്തിയാക്കി.
'രാജാ ഹരിശ്ചന്ദ്ര സിനിമയുടെ വിജയത്തെ തുടർന്ന് നാസിക്കിൽ ഒരു ഫിലിം സ്റ്റുഡിയോ സ്ഥാപിച്ചു.
ഇന്ത്യൻ സിനിമയുടെ പിതാവ് എന്നു വിശേഷിപ്പിക്കപ്പെടുന്ന ദാദാ ഫാൽക്കെ 1944 ഫിബ്രവരി 16ന് അന്തരിച്ചു.
ഇന്ത്യൻ സിനിമയ്ക്ക് സമഗ്ര സംഭാവനകൾ നൽകുന്നവർക്ക് ഭാരത സർക്കാർ നൽകുന്ന അവാർഡാണ് ദാദാ ഫാൽക്കെ അവാർഡ്'
ആദ്യമായി ദാദാസാഹബ് ഫാൽക്കെ അവാർഡ് നേടിയത് ദേവികാറാണി റോറിച്ച് ആണ്(1969)
ആദ്യമായി ദാദാസാഹിബ് ഫാൽക്കെ അവാർഡു നേടിയ മലയാളി അടൂർ ഗോപാലകൃഷ്ണനാണ്(2004).
