ഫ്രഞ്ചുകാർ - french east india company

ഫ്രഞ്ച് ഈസ്റ്റ് ഇന്ത്യാ കമ്പനി സ്ഥാപിതമായത്?

ans : 1664

ഫ്രഞ്ച് ഈസ്റ്റിന്ത്യാ കമ്പനി സ്ഥാപിക്കുന്ന സമയത്തെ ഫ്രഞ്ച് ചക്രവർത്തി?

ans : ലൂയി XIV

ഫ്രഞ്ച് ഈസ്റ്റ് ഇന്ത്യാ കമ്പനി സ്ഥാപിതമായത്?

ans : 1664

ഫ്രഞ്ച് ഈസ്റ്റിന്ത്യാ കമ്പനി സ്ഥാപിക്കുന്ന സമയത്തെ ഫ്രഞ്ച് ചക്രവർത്തി?

ans : ലൂയി XIV

1668-ൽ ഫ്രഞ്ചുകാർ ഇന്ത്യയിൽ ആദ്യത്തെ വ്യാപാര കേന്ദ്രം ആരംഭിച്ചത്?

ans : സൂററ്റിൽ

ഫ്രഞ്ചുകാരുടെ പ്രധാന ഫാക്ടറികൾ?

ans : സൂററ്റ്,മസൂലി പട്ടണം, ചന്ദ്രനഗർ, പോണ്ടിച്ചേരി

ഇന്ത്യയിലെ പ്രധാന ഫ്രഞ്ച് താവളങ്ങൾ?

ans : മാഹി, കാരയ്ക്കൽ, യാനം, ചന്ദ്ര നഗർ,പോണ്ടിച്ചേരി

ഫ്രഞ്ചകാരുടെ കേരളത്തിലെ വ്യാപാര കേന്ദ്രം സ്ഥിതി ചെയ്തിരുന്നത്?

ans : മാഹി (മയ്യഴി)

ഇന്ത്യയിലെ ‘ഇംഗ്ലീഷ് ചാനൽ’ എന്നറിയപ്പെടുന്നത്?

ans : മയ്യഴിപ്പുഴ

മയ്യഴിപ്പുഴ (മാഹിപ്പുഴ)യ്ക്ക് ഇംഗ്ലീഷ് ചാനൽ എന്ന് പേരുവരാൻ കാരണം?

ans : ഫ്രഞ്ച് അധീന പ്രദേശമായിരുന്ന മാഹിയെ ബ്രിട്ടീഷ് അധീന പ്രദേശങ്ങളിൽ നിന്നും വേർതിരിച്ചിരുന്നത് മയ്യഴിപ്പുഴയാണ്

ഫ്രഞ്ചുകാർ ഇന്ത്യ വിട്ടുപോയ വർഷം?

ans : 1954

ഇന്ത്യയിലെ ഫ്രഞ്ചുകാരുടെ ആസ്ഥാനം?

ans : പോണ്ടിച്ചേരി

പോണ്ടിച്ചേരിലെത്തിയ ആദ്യ ഫ്രഞ്ച് ഗവർണർ?

ans : ഫ്രാങ്കോയി മാർട്ടിൻ

പോണ്ടിച്ചേരിയുടെ പിതാവ് എന്നറിയപ്പെടുന്നത്?

ans : ഫ്രാങ്കോയി മാർട്ടിൻ

യൂറോപ്പിൽ ഇംഗ്ലണ്ടിനേയും ഫ്രാൻസിനേയും വേർതിരിക്കുന്ന ചാനൽ?

ans : ഇംഗ്ലീഷ് ചാനൽ

ഇംഗ്ലണ്ടിനേയും ഫ്രാൻസിനേയും വേർതിരിക്കുന്ന കടലിടുക്ക്?

ans : ഡോവർ കടലിടുക്ക്

‘പരന്ത്രീസുകാർ’ എന്നറിയപ്പെട്ടിരുന്നത്?

ans : ഫ്രഞ്ചുകാർ

ഇന്ത്യയിലെ ഫ്രഞ്ച് പതനത്തിന് കാരണമായ യുദ്ധം?

ans : വണ്ടിവാഷ് യുദ്ധം

വണ്ടിവാഷ് യുദ്ധം നടന്ന വർഷം?

ans : 1760

വണ്ടിവാഷ് യുദ്ധത്തിന്റെ ഫലമായി ഉണ്ടാക്കിയ സന്ധി?

ans : പാരീസ് ഉടമ്പടി(1763)

വണ്ടിവാഷ് യുദ്ധത്തിൽ പരാജയപ്പെട്ട ഫ്രഞ്ച് സൈന്യാധിപകൻ?

ans : കൗണ്ട് ഡി ലാലി
