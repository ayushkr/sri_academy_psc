ബ്രിട്ടീഷ് ആധിപത്യം - ഗവർണർ ജനറൽമാർ -

British Governor Generals- lord ellenborough

എല്ലൻബെറോ(1842-1844)

സിന്ധ് മേഖല ബ്രിട്ടീഷ് ഇന്ത്യയോട് കൂട്ടിചേർത്ത ഗവർണർ ജനറൽ?

ans : എല്ലൻബെറോ പ്രഭു

ഇന്ത്യയിൽ അടിമത്തം നിയമവിരുദ്ധമായി പ്രഖ്യാപിച്ചത്?

ans : 1843

അടിമത്തം നിയമവിരുദ്ധമാക്കിയ ഇന്ത്യൻ ഗവർണർ ജനറൽ?

ans : എല്ലൻബെറോ പ്രഭു
